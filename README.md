<h1 align="center">ms-editor</h1>

forked from [canvas-editor](https://github.com/Hufe921/canvas-editor)

## install from npm

```bash
npm i @qinsuyan/ms-editor
```

## 更改内容

* 表格可指定边框宽度

* 增加变量类型
    * 文字
    * 图片

* 增加循环功能

* 增加绘制横线和箭头

* 增加文本框

## install

`pnpm install`

## dev

`npm run dev`

## build

#### app

`npm run build`

#### lib

`npm run lib`
